Raspberry Pi User Input

Posted 10th February 2017

After hours of rigorous troubleshooting, I finally was able to code a python script, max.py, that allows the user to toggle the LED on and off and run a special morse code message all at the push of a button. When the "w" button is pressed, the program turns on the led with the "GPIO.output(18,GPIO.HIGH)" command; When the "s" button is pressed, the program turns on the led with the "GPIO.output(18,GPIO.LOW)". When the "q" button is pressed, the program runs the morse code message from the previous challenge.
